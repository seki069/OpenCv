# Glute Project

## 概要

このプロジェクトでは、OpenCVとMediaPipeを使用して、動画ファイルを処理し、グルートブリッジ（大臀筋のトレーニング）デモを行います。プログラムは動画内の動きを解析し、結果を可視化します。

## 必要なライブラリ

- **OpenCV**: 画像処理ライブラリ
- **MediaPipe**: 動作解析用ライブラリ

- 必要なライブラリは以下のコマンドでインストールできます。

bash
pip install opencv-python mediapipe
